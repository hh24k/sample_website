Important HTML Tags

This page is designed as a quick reference for some of the common tags covered in Learn Enough HTML to Be Dangerous. In the process of making it, we will learn how to make HTML tables> via table and related tags.

The tales below don't include all the HTML tags, but they do include many of the most important HTML tags.

Block Tags

Tag Name Purpose
h1–h6 headings include a heading (level 1 – level 6)
p paragraph include a paragraph
table talbe include a table
tr table row include a row of data
td table data include a table data cell
div division define block level section in document
header header lable the page header
ol ordered list list elements in numberical order
ul unordered list list emlements whose order doesn't matter
blockquote blockquote show formated quotation
br break enter line break

Inline Tags

Tag Name Purpose Example Result
em emphasized make emphasized text <em>techical sophistication</em> techical sophistication
strong strong make strong text <strong>at least a billion<strong> at least a billion people
a anchor make hyperlink <a href="#" learnenough </a> learnenough
img image include an image <img src="kitten.jpg" alt="a kitten">
code code format as source code <code>table</code> table
span span define inline section in document <span style="color: red;">Call me Ishmael.</span> Call me Ishmael.
